설치방법

    1. cmd 창에서 관리자 권한으로 실행
        npm instal pm2 -g

    2. window powershell에서 관리자 권한 실행
        Set-ExecutionPolicy RemoteSigned


사용방법

    1. 특정 프로젝트 실행
        pm2 start 파일명.확장자

        - 프로젝트를 강제종료하여도 자동 재시작됨


    2. 현재 실행 중인 pm2 모니터링
        pm2 monit


    3. 현재 실행 중인 프로젝트 목록
        pm2 list


    4. 실행 중인 특정 프로젝트 종료
        pm2 stop 프로젝트명


    5. 프로젝트 실시간 업데이트 확인
        pm2 start 파일명.확장자 --watch
        
        - 프로젝트 파일을 업데이트 하면 새로고침 시 바로 적용


    6. pm2 로그 확인
        pm2 log
        
        - 프로젝트 실시간 업데이트 시 프로젝트 변경여부 확인 위해 필요
